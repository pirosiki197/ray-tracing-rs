# Ray Tracing in One Weekend

![](images/random_spheres.png)

## Original work
- Original Authors: Peter Shirley, Trevor David Black, Steve Hollasch
- Original Site: https://raytracing.github.io/

## How to Run
```rust
cargo run --release > image.ppm
```
